## 🚀 IMMEDIATE SETUP

### For Windows (Recommended)
```
1. Open Command Prompt in the project folder
2. Run: check-prerequisites.bat
3. Run: setup-database.bat
4. Run: start-backend.bat (Keep this running)
5. Open new Command Prompt, Run: start-frontend.bat
6. Visit: http://localhost:3000
```

### For Mac/Linux
```
1. Open Terminal in project folder
2. Run: bash check-prerequisites.sh
3. Run: mysql -u root -p < database/schema.sql
4. Run: bash start-backend.sh (Keep this running)
5. Open new Terminal, Run: bash start-frontend.sh
6. Visit: http://localhost:3000
```

## 🆘 QUICK TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| Backend won't start | Check MySQL is running, verify credentials |
| Frontend can't connect | Check backend is on port 8080, clear cache |
| Port already in use | Change port in config file |
| Database won't create | Check MySQL password, run as admin |
| npm install fails | Delete node_modules, try again |

See `README.md` for detailed troubleshooting.
